HTMLManualEditor 操作マニュアル

このマニュアルでは、HTMLManualEditorの基本的な操作方法を説明します。
本エディタは、HTML形式のマニュアルやドキュメントを直感的に作成・編集できるツールです。

このマニュアル自体がHTMLManualEditorで作成されています。編集を試すことで操作を体験できます。
元に戻したい場合は Ctrl+Z または「元に戻す」ボタンをご利用ください。

1. 画面構成

画面は以下の5つのエリアで構成されています。

エリア 説明
ツールバー（上部） ファイル操作、書式設定、要素の挿入、図形、要素の操作、ページ設定、ズームなどのボタンが並んでいます。
要素ツリー（左側） ページ内の要素をツリー形式で表示します。要素をクリックして選択できます。パネルは折りたたみ可能です。
プレビュー（中央） 編集中のHTMLをリアルタイムにプレビュー表示します。直接クリックして要素を選択・編集できます。ページ区切り位置も表示されます。
プロパティ（右側） 選択中の要素のスタイル（色、サイズ、余白など）を変更できます。ページ全体や図形の専用プロパティも表示されます。
ステータスバー（下部） バージョン情報と操作ヒント（「クリックで選択」「ダブルクリックでテキスト編集」など）が表示されます。

2. ファイル操作

新規作成

ツールバーの「新規」ボタンをクリックすると、空の新規ドキュメントを作成します。未保存の変更がある場合は確認メッセージが表示されます。

HTMLファイルの読み込み

「読込」ボタンをクリックし、HTMLファイル（.html / .htm）を選択すると、エディタに読み込まれます。

HTMLファイルの保存

「保存」ボタンをクリックすると、現在の編集内容をHTMLファイルとしてダウンロードします。

PDF出力

「PDF」ボタンをクリックすると、ブラウザの印刷機能を使ってPDFとして出力できます。

ファイル名はツールバー右側に表示されます。未保存の変更がある場合はファイル名の後に「*」が付きます。

3. 要素の選択と編集

要素を選択する

プレビュー上の要素をクリックすると、その要素が選択されます。選択された要素は青い枠線で囲まれます。

  • 要素ツリーからも選択できます
  • プレビューの外側（灰色の余白部分）をクリックすると、ページ全体を選択できます
  • 配置した要素より下の余白エリアをクリックすると、最後の要素が選択されます

テキストを編集する

要素をダブルクリックすると、テキスト編集モードになります。文字を直接入力・編集できます。

元に戻す／やり直し

ツールバーの矢印ボタン、または Ctrl+Z（元に戻す）、Ctrl+Y（やり直し）で操作を取り消し・復元できます。

4. 要素の挿入

ツールバーに並ぶアイコンで、さまざまな要素をページに追加できます。

クリックで挿入

要素を選択した状態でツールバーのアイコンをクリックすると、新しい要素が追加されます。

  • ブロックやカラムなどの入れ物要素を選択中 → その中に追加されます
  • 通常の要素を選択中 → 選択中の要素の下に追加されます
  • 何も選択していない場合 → ページの末尾に追加されます

ドラッグ＆ドロップで挿入

ツールバーのアイコンをドラッグしてプレビュー上の好きな位置にドロップすると、その位置に要素を挿入できます。

  • 要素の上半分にドロップ → その要素の前に挿入
  • 要素の下半分にドロップ → その要素の後に挿入
  • 入れ物要素の中央にドロップ → その中に挿入
  • ページ下部の空白にドロップ → 末尾に追加

挿入できる要素一覧

アイコン 要素名 説明
H1 見出し1 最も大きな見出し。ページタイトルなどに使用します。
H2 見出し2 セクションの見出し。章立てに使用します。
H3 見出し3 小見出し。小セクションの区切りに使用します。
T 段落 本文テキストの段落を挿入します。
画像 画像 クリック：ファイル選択で画像を挿入。ドラッグ：画像の配置枠を挿入（後からクリックで画像を選択）。
表 テーブル 3列×2行の表を挿入します。行・列の追加はテーブル選択後に操作できます。
□ ブロック 装飾付きのコンテンツ領域を挿入します。補足情報や注意書きに便利です。
リンク リンク クリック可能なリンクテキストを挿入します。
― 水平線 セクション間の区切り線を挿入します。
2列 2カラム 左右2つに分かれたレイアウト。各カラム内に要素を配置できます。
3列 3カラム 3つに分かれたレイアウト。各カラム内に要素を配置できます。
図形 図形 丸・三角・四角・矢印や番号付き図形を挿入します。詳細はセクション8を参照。

5. 要素の並べ替え・操作

ドラッグで並べ替え

プレビュー上で要素をクリックして選択した後、そのままドラッグすると、要素の位置を入れ替えることができます。ドロップ先には青い目印ラインが表示されます。

ボタンで操作

要素を選択した状態で、ツールバーの以下のボタンが使えます。

ボタン 機能
コピー 選択中の要素を複製します。
↑（上移動） 選択中の要素を1つ上に移動します。
↓（下移動） 選択中の要素を1つ下に移動します。
削除 選択中の要素を削除します。

6. 書式設定

要素を選択した状態で、以下の書式をツールバーから適用できます。

ボタン 機能
B（太字） 選択中の要素のテキストを太字にします。
I（斜体） 選択中の要素のテキストを斜体にします。
U（下線） 選択中の要素のテキストに下線を付けます。
左揃え／中央揃え／右揃え／両端揃え テキストの配置を変更します。
箇条書き／番号付きリスト リスト形式のテキストを挿入します。

7. テーブルの操作

テーブルには専用の操作機能があります。

テーブル全体を選択する

テーブルの左上角付近に表示されるハンドル（つまみ）をクリックすると、テーブル全体を選択できます。このハンドルをドラッグすると、テーブルの位置を移動できます。

行・列の追加

テーブルにマウスを乗せると、右端と下端に「＋」ボタンが表示されます。

  • 右端の「＋」 → 列を追加
  • 下端の「＋」 → 行を追加

テーブルまたはセルを選択した状態で、右側のプロパティパネルにも「行追加」「行削除」「列追加」「列削除」のボタンが表示されます。

行・列の並べ替え

テーブル内の行や列はドラッグで並べ替えることができます。行をドラッグすると目印ラインが表示され、ドロップした位置に行が移動します。列も同様にドラッグで順序を変更できます。

セルの編集

セルをダブルクリックすると、テキストを直接編集できます。

8. 図形の挿入と編集

ツールバーの「図形」ボタンをクリックすると、図形メニューが開きます。

挿入できる図形

種類 図形
基本図形 丸、三角、四角、矢印
番号付き図形 ①〜⑨（番号入りの丸）

図形の挿入方法

  • クリック：図形メニュー内のアイコンをクリックすると、ページ上に図形が配置されます。
  • ドラッグ：図形メニュー内のアイコンをドラッグして、プレビュー上の好きな位置にドロップできます。

図形の移動

図形はページ上で自由に配置できます（絶対配置）。図形をクリックして選択した後、ドラッグすると位置を移動できます。

図形のプロパティ

図形を選択すると、右側のプロパティパネルに図形専用の設定が表示されます。

項目 説明
塗り色 図形の塗りつぶし色を変更します。
線の色 図形の輪郭線の色を変更します。
線の太さ 輪郭線の太さを変更します。
回転 スライダーで0〜359度の範囲で図形を回転できます。
位置（上・左） 図形の配置位置を数値で指定します（例：50px）。
サイズ（幅・高さ） 図形の大きさを変更します（例：60px）。
図形は他の要素の上に重ねて表示されます。手順書の番号付けや、注目箇所の強調マークなどに活用できます。

9. 画像の挿入

直接挿入（クリック）

ツールバーの「画像」アイコンをクリックすると、ファイル選択ダイアログが開きます。画像を選択すると、即座にプレビューに挿入されます。

配置枠を使った挿入（ドラッグ）

「画像」アイコンをドラッグ＆ドロップすると、配置枠（プレースホルダー）が挿入されます。配置枠をクリックすると、ファイル選択ダイアログが開き、画像を選択できます。

画像はカラムレイアウト（2列・3列）の中にも配置できます。親要素の幅に合わせて自動的にサイズが調整されます。

10. カラムレイアウト（入れ子構造）

2列・3列のカラムレイアウトやブロック要素の中に、別の要素を配置できます。

配置方法

  • クリック挿入：カラムの片側を選択した状態で、ツールバーのアイコンをクリック
  • ドラッグ挿入：ツールバーからアイコンをドラッグし、カラム内にドロップ
制限事項：入れ子は2階層まで（親子関係）です。孫要素（入れ物の中の入れ物の中に要素）の配置はできません。

11. プロパティパネル

要素を選択すると、右側のプロパティパネルでスタイルを細かく調整できます。

レイアウト

項目 説明
幅・高さ 要素のサイズを指定します（例：100%、200px、auto）。
マージン (上/右/下/左) 要素の外側の余白を個別に調整します。
パディング (上/右/下/左) 要素の内側の余白を個別に調整します。
Flex (子要素比率) カラムレイアウト内の子要素の幅比率を設定します（例：1、2）。

テキスト

項目 説明
フォントサイズ 文字の大きさを変更します（例：16px）。
太さ 文字の太さを選択します（normal、bold、100〜900）。
文字色 カラーピッカーまたはテキスト入力で色を指定します。
揃え テキストの配置方向（左、中央、右、両端）を変更します。
行間 行と行の間隔を設定します（例：1.5、2.0）。
字間 文字と文字の間隔を設定します（例：1px、normal）。
フォント フォントを選択します（メイリオ、ヒラギノ角ゴ、游ゴシック、MSPゴシック、MS P明朝、Arial、Times New Roman、等幅）。

背景・ボーダー

項目 説明
背景色 カラーピッカーで要素の背景色を選択します。
ボーダー幅 枠線の太さを指定します（例：1px）。
ボーダー色 枠線の色をカラーピッカーで選択します。
ボーダー形 枠線の種類（なし、実線、破線、点線、二重線）を選択します。
角丸 角の丸みを設定します（例：8px）。

Flex コンテナ設定

カラムレイアウト（2列・3列）などのFlex要素を選択すると、以下のFlexコンテナ設定が表示されます。

項目 説明
方向 子要素の並び方向を変更します（横、縦、横逆、縦逆）。
間隔 (gap) 子要素間の間隔を設定します（例：20px）。
主軸揃え 主軸方向の子要素の配置（先頭、中央、末尾、均等）を設定します。
交差軸揃え 交差軸方向の子要素の配置（伸縮、先頭、中央、末尾、ベースライン）を設定します。

要素固有のプロパティ

選択した要素の種類に応じて、以下の専用プロパティが表示されます。

  • 画像（img）：画像URL（src）、代替テキスト（alt）
  • リンク（a）：リンクURL（href）
  • テーブル関連：行追加・行削除・列追加・列削除のボタン

12. ページ設定

ページの向き

ツールバー右側の「A4縦」/「A4横」ボタンをクリックすると、ページの向きを切り替えられます。

向き サイズ
A4縦（ポートレート） 幅 794px × 高さ 1123px
A4横（ランドスケープ） 幅 1123px × 高さ 786px

ページ全体のスタイル

プレビューの余白部分（要素の外側）をクリックすると、ページ全体が選択されます。プロパティパネルにページ専用の設定が表示されます。

項目 説明
背景色 ページ全体の背景色を変更します。
文字色 ページ全体のデフォルト文字色を変更します。
フォントサイズ ページ全体のデフォルトフォントサイズを変更します。
行間 ページ全体のデフォルト行間を変更します。
フォント ページ全体のデフォルトフォントを選択します。
パディング (上/右/下/左) ページ内側の余白を調整します。
マージン (上/右/下/左) ページ外側の余白を調整します。

13. ズーム

ツールバー右側の「−」「＋」ボタンでプレビューの表示倍率を変更できます（25%〜200%、10%刻み）。現在の倍率はボタンの間に表示されます。

14. キーボードショートカット

キー操作 機能
Ctrl+Z 元に戻す
Ctrl+Y やり直し
Delete / Backspace 選択中の要素を削除（テキスト編集中は文字を削除）
ヒント：このマニュアルを参考にしながら、実際に要素の追加・編集・並べ替えを試してみてください。操作に慣れると、さまざまなレイアウトのHTMLドキュメントをすばやく作成できるようになります。
